# AUTOMATED_REPORT_GENERATION]

## Project Overview
This project demonstrates **Automated Report Generation** using Python.  
The script reads student performance data from a CSV file, analyzes the data, and automatically generates a **formatted PDF report** using the **ReportLab** library.

This project is developed as part of the **CODTECH Internship Task** on Automated Report Generation.

---

## Objectives
- Read structured data from a file (CSV)
- Perform data analysis on student performance records
- Generate a professional PDF report automatically
- Demonstrate automation using Python libraries

---

## Input Data
The input data is stored in a CSV file named `data.csv`.

### Sample Columns:
- **StudentName** – Name of the student
- **Department** – Department or branch
- **Marks** – Student marks
